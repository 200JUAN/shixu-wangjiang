# 时叙万疆
## 项目简介
时叙万疆——贵州少数民族传统节日时序展示平台，集成民族历法、民俗图谱、AI民族助手，记录民族节庆时序文化。

## 技术栈
前端：HTML + CSS + JavaScript
后端：Python Flask

## 核心功能模块
1. 时序历馆：公历农历互转，全年少数民族节日时序查询
2. 百族图谱：各民族图文介绍、传统民族纹样展示
3. 风物叙录：民族服饰、民俗、传统建筑详情阅览
4. AI民族助手：民俗问答、语音交互讲解民族文化
5. 用户系统：账号登录、节日收藏、定时邮件推送

## 本地运行教程
1. 将仓库克隆至本地文件夹
2. 安装Python Flask依赖库
3. 启动后端Flask服务
4. 浏览器访问本地地址即可打开网页

## 项目关键词
贵州少数民族、民族日历、传统节庆、民俗文化、前后端网页项目
